# leonard_riverpod

A reference extension for [Leonard](https://github.com/memento-engineering/lenny) — reports which
Riverpod providers are currently loading, as an observation fragment and a stability signal.

Add it to a Flutter app that already installs `leonard_flutter`, then register it with the binding.
See the [Leonard README](https://github.com/memento-engineering/lenny#write-an-extension) for usage.
